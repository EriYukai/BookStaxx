BookStaxx 설정

북마크 바튼 설정

북마크 버튼 표시
북마크 바를 열고 닫는 버튼을 표시합니다
버튼 위치
북마크 버튼이 표시될 위치를 선택합니다
버튼 크기
북마크 버튼의 크기를 조정합니다
사용자 지정 크기
북마크 버튼의 크기를 세밀하게 조정합니다
버튼 색상
북마크 버튼의 배경 색상을 설정합니다
텍스트 색상
버튼 내 텍스트 색상을 설정합니다
커스텀 이미지
버튼에 커스텀 이미지를 사용합니다 (URL 입력)

북마크 바 설정

북마크 바 표시
북마크가 표시되는 바를 화면에 표시합니다
북마크 바 위치
북마크 바가 표시될 위치를 선택합니다
위치 시각적 선택
북마크 바 위치를 시각적으로 선택합니다
바 높이
북마크 바의 높이를 조정합니다
배경 색상
북마크 바의 배경 색상을 설정합니다
텍스트 색상
북마크 항목의 텍스트 색상을 설정합니다
북마크 바 표시 애니메이션
북마크 바가 처음 표시될 때의 애니메이션 효과를 선택합니다
북마크 바 모양
북마크 바의 모양과 레이아웃을 선택합니다
북마크 텍스트 표시
북마크 항목에 텍스트(이름)를 표시할지 여부를 설정합니다
텍스트 길이 제한
북마크 이름의 최대 표시 길이를 설정합니다 (0: 제한 없음)
크롬 UI 스타일 사용
북마크바를 크롬 브라우저의 기본 북마크바와 유사하게 표시합니다
크롬 북마크 불러오기
크롬 기본 북마크를 BookStaxx 북마크바로 불러옵니다
북마크바 유지 설정
페이지 이동 시 북마크바를 유지합니다. 북마크바가 사라지지 않고 계속 표시됩니다.
