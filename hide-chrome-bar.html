Chrome 북마크 바 숨기기 안내

방법 1: 키보드 단축키 사용 (가장 빠른 방법)

1
Chrome 브라우저가 활성화된 상태에서 Ctrl + Shift + B 키를 동시에 누릅니다.
2
이 단축키를 누를 때마다 북마크 바가 표시되거나 숨겨집니다.
참고: 이 단축키는 Chrome 브라우저의 기본 단축키로, BookStaxx 확장 프로그램이 없어도 사용할 수 있습니다.

방법 2: Chrome 설정에서 변경

1
Chrome 브라우저 오른쪽 상단의 점 세 개(⋮) 메뉴를 클릭합니다.
2
메뉴에서 설정을 클릭합니다.
3
왼쪽 메뉴에서 모양을 클릭합니다.
4
북마크 바 표시 옵션을 끕니다.

BookStaxx 사용 팁

Chrome 북마크 바를 숨긴 후에는 BookStaxx 북마크 바만 표시되어 더 깔끔한 브라우징 경험을 즐길 수 있습니다.

알림: Chrome의 보안 정책으로 인해 확장 프로그램에서 직접 Chrome 북마크 바를 숨길 수 없습니다. 사용자가 직접 위의 방법 중 하나를 사용하여 숨겨야 합니다.
